# MonitorApp

**MonitorApp** — это демонстрационный проект на ASP.NET Core MVC, реализующий фоновую задачу сбора показателей производительности (например, использования памяти) и отображающий их в веб-интерфейсе.

### Поведение приложения

- Каждые **5 секунд** фоновый сервис добавляет новую запись в `MonitorPanel`.
- Все записи сортируются по убыванию `Id` и отображаются в таблице.

### В таблице отображаются:

- **Номер записи**
- **Время фиксации**
- **Объём памяти в мегабайтах**
